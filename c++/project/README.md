# 👆🏻프로젝트 개요

<aside>
✅ 콘솔 창에서 텍스트 기반으로 플레이 가능한 여러가지 게임들을 만들었습니다.

</aside>

- 언어

  - C++

- 개발 환경

  - Windows
  - Visual Studio

- 개발 기간

  - 2023/09/13 ~ 2023/09/20

- 회고

  - 얻은 경험

    - 난수 생성
    - 2차원 배열
    - Class, 상속, 업캐스팅 활용
    - 파일 분리

  - 보완 사항
    - User Interface 개선
    - 반복되는 코드 함수화
    - 직접 입력된 하드 코드들을 변수화 하여 소프트 코딩화
